# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-13

### Added
- End-to-end computer vision pipeline for palographic analysis.
- Desktop app with hybrid workflow (manual inputs override automatic extraction).
- CLI execution flow with optional ML fusion (`assist`, `hybrid`, `override`).
- Margin inversion toggle for mirrored images (`swap_lr_margins`).
- ML tooling: dataset builder, trainer and benchmark scripts.
- Automated full process script with real local examples.
- GitHub-ready documentation, contribution guide, and CI workflow.

### Notes
- ML benchmark currently includes pseudo-labeled evaluation for internal consistency.
- Human review remains required for critical psychometric interpretation.
