## Version 1 (technical + impact)

Hoje publiquei o **Palo Analyzer**, um projeto em Python para automatizar a correção do Teste Palográfico com foco em **auditabilidade** e **precisão prática**.

### O que construí
- Pipeline de visão computacional clássica com OpenCV (homografia, binarização, morfologia, detecção de traços).
- Extração de métricas psicométricas (total, NOR, inclinação, margens, espaçamentos, organização).
- Motor de regras em código com identificadores de regra e saída estruturada.
- Interface desktop para uso por não técnicos.
- Fluxo híbrido: entrada manual tem prioridade sobre análise automática.

### Diferenciais de engenharia
- Validação com benchmark de consistência.
- Modo de revisão humana para casos de baixa confiança.
- CI com testes automatizados para regras críticas.
- Projeto empacotável para distribuição em Windows.

Repositório: https://github.com/Iv4nLanna/Palo-Analyzer

Aberto a feedback técnico, colaboração e oportunidades em engenharia de software, visão computacional e data/ML.
